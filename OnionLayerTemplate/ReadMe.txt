To build and use vsextension:
	1. Select the following:
		OnionLayer.vstemplate
		Layer.Tests
		Layer.Models
		Layer.Interfaces
		Layer.Implementations
	2. Zip the selected files
	3. Place the zip into Documents\VisualStudio 2017\Templates\ProjectTemplates
	4. Right click an existing solution and add new project.  Select OnionLayer.
	5. Name your project the name of your layer, eg. Service or Data.
